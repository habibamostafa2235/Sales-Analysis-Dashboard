# 📊 Sales Analysis Interactive Dashboard (Excel)

## 📌 Project Overview
An interactive and comprehensive multi-tab Excel dashboard designed to analyze global sales data, customer performance, and product distributions. The goal of this project was to transform raw transactional data into actionable insights to support strategic business decisions.

---

## 📈 Key Metrics & Dataset Insights
The analysis was performed on a dataset containing over **2,800+ transaction rows**, revealing the following highlights:
- 💰 **Total Sales:** $10,032,628.85
- 📦 **Total Unique Orders:** 307
- 🤝 **Active Customers:** 92 global clients
- 🏷️ **Product Variety:** 109 unique products

---

## 🔍 Key Findings
- 🚗 **Top Product Line:** *Classic Cars* is the primary revenue driver, generating over **$3.9M** in sales.
- 🌍 **Top Geographic Markets:** The *USA* leads global sales with **$3.6M**, followed by Spain and France.

---

## 🗂️ Dashboard Structure & Features
The workbook is organized into distinct, user-friendly sheets:
1. 🏠 **Homepage:** A central navigation hub for the user.
2. 📊 **General / Executive Summary:** High-level KPIs and overarching sales trends.
3. 👥 **Customer Analysis:** Deep dive into client purchase history and deal sizes.
4. 🚀 **Sales Performance:** Breakdown of transactions by product line, country, and time.

---

## 🛠️ Technical Skills Applied
- 🧼 **Data Cleaning:** Handling missing values, formatting date fields, and ensuring transactional consistency.
- 🧮 **Data Aggregation:** Using **Pivot Tables** and advanced Excel formulas.
- 🎨 **Dynamic Visualization:** Implementing interactive **Slicers**, customized charts, and consistent color palettes.
